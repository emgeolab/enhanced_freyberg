# Enhanced Freyberg Model

Enhanced Freyberg model implemented with MF6 (FloPy) and PEST++ (pyEMU).

## Notebooks

Run the notebooks in order:

1. Build synthetic truth model
2. Set up PEST++
3. Calibrate with PEST++ GLM
4. Analyze uncertainty
5. Run posterior Monte Carlo

## Requirements

* MODFLOW 6
* PEST++
* FloPy
* pyEMU
* Python, NumPy, Pandas, Matplotlib
